# Welcome to Qxel 👋

<p align="center">
  <strong>The Indian Digital Marketplace for Creators.</strong>
</p>

Qxel is a digital marketplace where creators can sell and customers can purchase digital products including templates, notes, ebooks, software, courses, design assets, AI resources, and more.

This repository contains public resources used by Qxel, including:

- 📚 Qxel Taxonomy (Categories & Topics)
- 🏷️ Category definitions
- 🗂️ Marketplace classifications
- 📖 Public documentation
- 🌱 Community-driven improvements

---
